\doxysubsection{Detailed Description}
Provides a generic interface for generating random complex numbers, integers, and characters. 

This module offers utilities for generating random values with specific properties\+:
\begin{DoxyItemize}
\item Random complex numbers within the unit circle.
\item Random integers within a user-\/specified range.
\item Random selection from an array of characters. 
\end{DoxyItemize}
