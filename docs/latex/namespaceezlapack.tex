\doxysubsection{Detailed Description}
Global module that specifies the other modules used in ezlapack. 

This module allows the user to import the {\ttfamily ezlapack} module as a whole, simplifying the process by not requiring individual imports of other modules. It consolidates the necessary modules used within the {\ttfamily ezlapack} library.

The {\ttfamily ezlapack} module includes\+:
\begin{DoxyItemize}
\item {\ttfamily \doxylink{namespacemodule__ezmatmul}{module\+\_\+ezmatmul}}\+: A module for matrix multiplication operations.
\item {\ttfamily \doxylink{namespacemodule__random}{module\+\_\+random}}\+: A module for random number generation and related utilities.
\end{DoxyItemize}

By using {\ttfamily ezlapack}, users can access all the functionality provided by the underlying modules without needing to import them separately.
