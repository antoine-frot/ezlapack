\doxysubsection{Detailed Description}
Generic interface for generating random complex numbers. 

The {\ttfamily \doxylink{interfacemodule__random_1_1random__complex}{random\+\_\+complex}} interface generates random complex numbers (single or double precision) within the unit circle. Supports scalars and arrays up to rank-\/2.
